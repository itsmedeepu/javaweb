1.super Keyword:

  • super Keyword is used to access the properties of Parent class or super class or base class.
  • Is-a relationship(inheritance) is Mandatory to use super Keyword.
  • super keywords can be used inside the Non-Static methods.
  • super keyword cannot be used inside the static methods.
  • super Keyword can be any line of method or constructor.
  • Note: When super class and subclass properties(data-members and Memberfunctions) are same then to acess superclass properties we make use of super Keyword.

2.super()

  • super() is used to call the constructor of superclass.
  • Is-a (inheritance) is Mandatory to use super() Keyword.
  • super() is used only inside the constructor.
  • super() should be first line executed inside the constructor.
  • for no-argument constructor or parametrized constructor super() with no arguments is executed implicitly by jvm.
  • If we have to call super() with arguments it is not a implict call rather we should call it explicitly.
  • Note: this() and super() cannot be used together in the same constructor

3.Constructor Chaining

Passing the values from sub class constructor to superclass constructor is known as constructor chaining.

3.Differences

SI.NO this super
1. this keyword is used to point the current instance(object). super keyword is used to access the properties of super class.
2. Is-a(inheritance) relationship is not required. Is-a relationship is Mandatory.
3. can be used in non-static methods & constructors not in static methods. can be used in non-static methods & constructors not in static methods.
4. It can be executed in any line of the method or constructor. It can be executed in any line of the method(or) constructor.
5. this is Implicit or explicit for non-static properties. super keyword is explicit.
SI.NO this() super()
1. this() is used to call constructor from another constructor. super() is used to call constructor of super class from sub class.
2. this() is used to pass value from one constructor to another constructor of same class. super() pass value from subclass to super class.
3. this() should be the first line of the contructor. super() should be first line of the constructor.
4. this() only used for constructors not in methods(static or non static). super() only used for constructors not in methods(static or non static).
5. It is known as constructor calling. It is known as constructor chaining
6. this() is explicit super() is implicit for default constructors for parametrized constructors super() is explicit.

5.final

  • final keyword in java is used to provide restrictions by jvm.
  • final keyword can be used for a variable,methods,classes.
  • final keyword cannot be used for static methods.
  • final keyword cannot be used for constructors.

final keyword

  • To convert variable into constant we use final keyword.
  • Once value is intialized to final variable cannot be re-assigned.
  • If the final variable is decalared it can be intialized only once.
  • final methods cannot be overriden. rather it can be extended from super class to subclass.

Final keyword for classes

classes with final keyword cannot be extended they are eligible to be sub class not super class.
